Пластины для печати BambuLab

У BambuLab 4 варианта пластин:

  • Cool Plate 
  • High Temp Plate
  • Engineering Plate
  • Textured PEI Plate

 

Cool Plate и High Temp Plate

Cool Plate и High Temp Plate представляют из себя наклейки для Engineering Plate.

 

Cool Plate

Cool Plate предназначена для PLA / PVA / TPU.

Температура эксплуатации - 30-35° C

Обязательно применение клея.

В первых выпусках на наклейке Cool Plateпроизводитель также указывал PETG и ABS, но довольно быстро осознал свою ошибку. Печатать этими материалами на ней можно, но есть риск вспучивания наклейки.

Прямых аналогов на али нет, ввиду узости применения. Продавцы и покупатели тяготеют к более универсальным решениям.

 

High Temp Plate 

High Temp Plate представляет из себя PEI-наклейку, и предназначена для всех материалов, которыми по мнению производителя может печатать их продукт. (PLA / ABS / PETG / TPU / ASA / PVA / PC / PA)

Температура эксплуатации - 35-110° C

Обязательно применение клея.

Аналогов на али масса, одни из самых популярных - Energetic.

 

Engineering Plate

Engineering Plate является как самостоятельной пластиной для печати, так и основой для наклеек Cool Plate и High Temp Plate, и представляет из себя стальной лист с неким покрытием (судя по поведению, какая-то эмаль; в первых выпусках была глянцевой, затем покрытие сделали немного шершавым).

Предназначена для печати TPU / PETG / ABS / PC / PA/

Температура эксплуатации - 35-110° C

Обязательно применение клея.

Аналогов не встречал.

 

Textured PEI Plate

Textured PEI Plate представляет из себя стальной лист с двухсторонним напылением из частиц PEI.

Предназначена для всех материалов, которыми по мнению производителя может печатать их продукт. (PLA / ABS / PETG / TPU / ASA / PVA / PC / PA)

Температура эксплуатации - 35-110° C

Применение клея рекомендовано для TPU / PC / PA.

Протирать ацетоном нельзя!

Аналогов на али опять же масса, одни из самых популярных - Energetic.
